# Lesson Commerce App 🚀

📦 **Live Demo:** https://lesson-commerce-app.vercel.app/

---

## 📝 概要  
Lesson Commerce App は、教育コンテンツをオンラインで販売・管理するための Next.js + Supabase ベースのWEBアプリケーションです。  

### コマンド

supabase のデータベースから型を生成

```bash
npm run db:types
```
